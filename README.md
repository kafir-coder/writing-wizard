
# Writing Wizard

A simple real-time app that moderates and gets statistics from a text document

## Run Locally

Clone the project

```bash
  git clone git@github.com:kafir-coder/writing-wizard.git
```

Go to the project directory

```bash
  cd writing-wizard
```

Run the whole Application

```bash
  docker-compose up -d 
```
